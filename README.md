# KUL ANNDL Exercise Sessions

Exercise sessions for **Artificial Neural Networks & Deep Learning** at KU Leuven.

## Session 2 — Associative Memories & Time Series Prediction

### Contents

| File | Description |
|:-----|:------------|
| `hopfield.py` | Enhanced Hopfield Network class with visualization and analysis tools |
| `hopfield.ipynb` | Exercises on Hopfield networks (2D, 3D, MNIST digits, capacity, basins) |
| `time_series.ipynb` | Exercises on time series prediction (MLP, RNN, LSTM comparison) |
| `hopfield.tex` | LaTeX source for the Hopfield theory and exercises guide |
| `timeseries.tex` | LaTeX source for the time series theory and exercises guide |
| `s2.tex` | Main LaTeX document (includes both sections) |
| `SantaFe.npz` | Santa Fe laser dataset |

### Interactive Demos

Three Streamlit demos are included in `demos/`:

| Demo | Run command |
|:-----|:------------|
| **Hopfield Basin Explorer** | `streamlit run demos/hopfield_basin_explorer/app.py` |
| **Modern Hopfield & Self-Attention** | `streamlit run demos/modern_hopfield_attention/app.py` |
| **Time Series Model Comparison** | `streamlit run demos/timeseries_explorer/app.py` |

All demos are also deployed at: https://huggingface.co/spaces/psanchezML/anndl-session2-demos
